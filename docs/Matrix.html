Class: Matrix

Matrix

new Matrix(context)

2D transformation matrix object initialized with identity matrix.

The matrix can synchronize a canvas context by supplying the context as an argument.

All values are handled as floating point values.

applyToArray(points) → {Array}

Apply current matrix to array with point objects. Returns a new array with point objects.

A point object is an object literal:

{x: x, y: y}

so an array would contain:

[{x: x1, y: y1}, {x: x2, y: y2}, ... {x: xn, y: yn}]
Parameters:

points Array

array with point objects
Returns:

interpolate(m2, t) → {Matrix}

Interpolate this matrix with another and produce a new matrix. t is a value in the range [0.0, 1.0] where 0 is this instance and 1 is equal to the second matrix. The value is not constrained.
